# react-game

## React + SSR(NextJS)

### Control:
WASD/Up, Down, Left, Right/mouse -  ship control
spase/mouse click - attack
1,2,3,4 - choice of weapons

### The game includes:
lifes and health bar
types of enemies
varieties of cards
difficulty levels
weapon types
ship level
player points
high score table
fullscreen mode
sound control and sound effects


RUN DEV SERVER:
cd client
```
yarn dev
```

BUILD PROD VERSION:
```
yarn build
```
